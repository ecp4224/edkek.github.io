About Me

The Basics

My name is Eddie Penta, and I am a polyglot programmer. I specialize in API libraries and network applications, but that's not the only thing I do. I have also created UI designs and games in OpenGL.

Open For All!

I believe in the idea of FOSS (Free and Open Source Software), so all of my projects are OpenSourced licensed either under MIT or GNU. I host all my open source projects on Github. However, when I'm working for a client, I keep the project private unless I get permission to Open Source it.

Education

I am currently attending Masconomet Regional Highschool and I'm in my senior year. I plan to graduate and go to college, majoring in Software Engineering. I have completed all the Computer Science courses that Masconomet has to offer (Computer Science I - Computer Science AP), and I am currently an assistant teacher for Computer Science III and Computer Science AP.

Programming Experience

I started programming back in 2009 when I wanted to create an Iphone app. After failing at trying to learn Objective-c, I moved on to an easier language which ended up being C#. I would start with simple plugins for a software to get my feet wet, then I worked my way up to applications and then to networking. After learning C#, I set out to learn as many programming languages as possible. Once I can do some form of reflection in the language or understand how the language works, I considered it mastered. Right now, I have mastered C# and Java, and I'm currently learning C++ along with HTML, CSS, PHP and Javascript.
