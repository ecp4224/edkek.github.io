About Me

My name is Eddie Penta and I am a polyglot programmer. I started programming when I was 12 years old, when I decided I wanted to make a game. I was determined to learn, so I went onto google and started teaching myself how to make a game. I was not successful then, and I had to overcome many failed attempts, but all those failed attempts were worth it. Before I got into actual programming, I would use tools such as MMF2 (Multimedia Fusion 2) and FPS Creator to create basic and fun projects. These tools helped me learn basic logic and problem solving, which helped build a foundation that would help me learn programming in later years.

When I was about 14, I started to get my feet wet in C# by making basic plugins/commands for minecraft classic servers. (If you unfamiliar with minecraft, its a Java based game where you build with blocks..and that's about it.) These plugins/commands functionality would range anywhere from funny messages to full utilities and mini-games. After about 2 years of this, my curiosity snuck up behind me once again and I decided I wanted to make my own minecraft classic server software. But not only did I want to make it a server software, I wanted to give it an API that was extensible and easy to use. The result is public on my github named 'GGS', and I consider it quite successful, even though it had it's faults. GGS was the start of my love for programming, and I learned many things from it including; networking, API design, plugin design, documentation, SQL, Java, GUI design, ect..

Recently, I graduated Masconomet Regional High school, completing all Computer Science classes (with the exception of Computer Science I). I even participated in the SVIP program during my senior year and became an assistant teacher for Computer Science 3 and Computer Science AP. Today, I am a freshmen at Wentworth Institute of Technology in Boston studying, of course, computer science. In my free time, I work on many public and private projects. Most of my public projects are 'just-for-fun' projects to get me to learn a certain technology or language.
